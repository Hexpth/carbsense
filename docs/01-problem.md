# Формальная постановка задачи

## Входы

В момент времени t₀ человек съел неизвестное количество углеводов C (g) с неизвестным типом (быстрые/медленные → характеризуется временем абсорбции τ_meal).

Доступны:
- CGM-измерения G(t) каждые 5 минут на интервале [t₀, t₀ + T], где T = 30..60 мин.
- История инсулина до t₀: список (t_i, dose_i), известна точно из помпы.
- Базовый профиль пациента: ICR, ISF, basal, DIA, вес.
- (Опционально) данные пульса/активности из Apple Health / Health Connect.

## Выходы

Оценка:
- C̃ — количество углеводов, граммы.
- τ̃ — характеристическое время абсорбции, минуты.
- σ_C — uncertainty (доверительный интервал ±σ).

## Метрики

Основная — **Mean Absolute Error**:

```
MAE = (1/N) Σ |C_i - C̃_i|
```

Дополнительные:
- **R²** для детектируемости тренда.
- **Recall@40g**: доля приёмов пищи >40 г, для которых C̃ > 25 г (т.е. алгоритм не пропустил большую порцию).
- **False positive rate**: при отсутствии еды (стресс-индуцированный hyperglycemia) алгоритм не должен выдавать C̃ > 15 г.
- **Latency** vs **accuracy** trade-off: насколько точно мы можем оценить за 15, 20, 25, 30, 45, 60 минут после t₀.

## Бейзлайны

Самый простой baseline — линейная регрессия:
```
C̃ = α · (G(t₀ + 30) - G(t₀)) + β
```
MAE ≈ 22–25 г на OpenAPS Data Commons. Любой ML-подход должен бить этот baseline.

Сильный baseline — Kalman Filter на упрощённой минимальной модели Бергмана. MAE ≈ 18 г.

## Чем задача сложна

1. **Inverse problem.** Прямая задача «C → G(t)» — нелинейная ODE с задержкой и фильтрацией (CGM измеряет интерстициальную жидкость с задержкой 5–15 мин). Обратная задача ill-posed.

2. **Confounders.** Инсулин уже введён до еды (pre-bolus) или после — он искажает кривую. Физическая активность, стресс, болезнь — всё меняет G(t).

3. **Inter-individual variability.** ICR и ISF у разных людей различаются в 5–10 раз. Модель, обученная на «среднем» пациенте, не работает на конкретном.

4. **Mixed meals.** Жир и белок замедляют абсорбцию углеводов (gastric emptying). Пицца поднимает сахар 4 часа, конфета — 30 минут. Двумерный параметр (C, τ) не всегда восстановим однозначно.

5. **Snacks / grazing.** Не все приёмы пищи — чёткие события. Перекусы, длинные ужины — размытые во времени.
